XML-RPC glue class¶

This module contains a “glue class” used by Twisted to export the NIPAP API functions over XML-RPC.

Each function call takes only one argument, an XML-RPC struct which contains the function arguments which are passed on to the Nipap-class.

For a detailed description of the API functions, see NIPAP API.

Authentication¶

Authentication is implemented using basic HTTP authentication. The user credentials are passed to the AuthFactory class together with the (optional) auth argument from the arguments-dict. If the authentication fails a HTTP 401 is returned.

See the Authentication library documentation for a detailed explanation of how the authentication system works.
